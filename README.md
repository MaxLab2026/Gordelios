# Gordelio's Food — Factura Semanal

Aplicación web para generar facturas semanales con PDF y WhatsApp.

## Deploy en Vercel

1. Sube esta carpeta a un repositorio en github.com
2. Ve a vercel.com/new e importa el repositorio
3. Click en Deploy — sin cambiar nada
4. Listo, obtienes una URL tipo gordelios-food.vercel.app

## Archivos

- index.html — App completa
- vercel.json — Configuración de Vercel
- .gitignore — Archivos ignorados por Git
- README.md — Este archivo

## Funcionalidades

- Facturación semanal organizada por día
- Generación de PDF profesional
- Envío por WhatsApp con resumen formateado
- Auto-guardado en localStorage
- Validaciones antes de exportar
- Responsive para móvil y escritorio
